\begin{frame}
\frametitle{Transmisión de la información}
\begin{itemize}
\item El análisis espacial obtiene nuevos datos a partir de datos espaciales ya existentes, y debe transmitirse la información que estos encierran.
\item Hay muchos tipos de información, muchas formas de extraerla y muchas formas distintas de transmitirla.
\end{itemize}
\end{frame}

\begin{frame}
\frametitle{El mapa como transmisor de información espacial}
\begin{itemize}
\item Un mapa no es siempre la mejor forma de transmitir una información con componente espacial.
\pause	
\item Un mapa es en sí un herramienta para el análisis espacial.
\pause	
\item El análisis espacial puede emplearse para mejorar un mapa (curvas de nivel, modelos de iluminación...).
\end{itemize}
\end{frame}
